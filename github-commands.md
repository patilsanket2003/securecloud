# GitHub Connection Commands

# Step 1: Add remote repository (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/securecloud.git

# Step 2: Push to GitHub
git push -u origin main

# Alternative: If you want to use SSH instead of HTTPS
# git remote add origin git@github.com:YOUR_USERNAME/securecloud.git
# git push -u origin main
